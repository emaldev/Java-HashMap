# Java-HashMap

This repository contains a comprehensive collection of my practical exercises focused on the `HashMap` data 
structure and the `Map` interface in Java. The goal of this collection is to demonstrate the progression from basic map
implementation to building interactive, console-based applications.

## Project Structure
The repository includes various modules covering different use cases:

* **StudentGradeManager.java**: The primary project; a dynamic system that takes student input (name/grade) via
* `Scanner`, processes the data, and categorizes students based on performance.
* **Student.java & StudentGrades.java**: Foundational files for managing and organizing student grade levels.
* **PhoneBookApp.java**: A practical implementation of `HashMap` focusing on adding and removing contact data.
* **ShapingManager.java**: An application of `Map` concepts to manage geometric shapes.
* **Main.java**: My introductory lesson files for understanding `HashMap` basics.
* **HelloWorde.java**: Initial experiments linking `HashMap` with the broader `Map` interface.

## Key Learning Outcomes
- **Dynamic Input**: Mastering user interaction using `Scanner` and handling input buffers.
- **Data Structures**: Efficiently using `put()`, `remove()`, and `Map.Entry` to manage key-value pairs.
- **Logical Categorization**: Implementing `if-else` branching logic to filter and classify data dynamically.
- **Organization**: Building modular code structures for different real-world scenarios like phone books and grade
-  management.
